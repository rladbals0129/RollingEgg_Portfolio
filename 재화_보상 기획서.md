## 1. 재화 시스템 개요

### 1.1. 재화 타입 정의

| 재화 타입 | 타입 별 기본 규칙 | 획득 방법 | 사용처 |
|-----------|-------------------|-----------|--------|
| **경험치** | • 알의 진화에 필요한 수치<br>• 플레이 타임 확보를 위하여 각 알의 러닝 게임 플레이 타임에 따라 경험치를 획득할 수 있다.<br>• 알 별로 각각 경험치를 가지며 지정한 최대 수치 100%에 도달 시 진화 후 리셋 된다.<br>• 경험치 100% 도달 시점의 알의 진화 스텟에 따라 확률적으로 최종 진화 외형이 결정된다.<br>• 진화 완료 후 해당 알의 경험치는 0으로 리셋되어 다시 육성 및 진화를 진행할 수 있다. | 각 알 별 러닝 게임 플레이 타임에 따라 획득 | 최대 경험치 도달 시 진화 |
| **공용 육성 재화** | • 알의 진화 스텟 획득을 위한 육성에 필요한 재화<br>• 모든 알의 내면 탐구(러닝 게임) 플레이 시 특정 색상이 아닌 선을 지나간 길이, 클리어 여부, 클리어 타임에 따라 획득한다.<br>• 모든 알의 육성 행동에 사용할 수 있다.<br>• 일정량의 공용 육성 재화를 사용하여 1회의 육성 행동 진행 시, 육성 행동 별 지정된 진화 스텟을 획득한다. | 1. 자동 달리기로 지나간 일반 선의 길이에 따라 획득<br>2. 게임 클리어 시 n배 획득<br>3. 클리어 타임 별 추가 획득<br>4. 중복 진화체 획득 시 재화 치환 | 모든 알 육성에 사용 |
| **전용 육성 재화** | • 특정한 알의 진화 스텟 획득을 위한 육성에 필요한 재화<br>• 특정한 알의 내면 탐구(러닝 게임) 플레이 시 특정 색상인 선을 지나간 길이에 따라 획득한다.<br>• 특정한 알의 육성 행동에 사용할 수 있다.<br>• 예: 노란 알 전용 재화는 노란 알의 육성 행동에만 사용할 수 있다.<br>• 일정량의 전용 육성 재화를 사용하여 1회의 육성 행동 진행 시, 육성 행동 별 지정된 진화 스텟을 획득한다. | 1. 특정 알로 게임 진행 시 지나간 특정 색상 선의 길이에 따라 획득<br>2. 중복 진화체 획득 시 재화 치환 | 특정 알 육성에만 사용 |

### 1.2. 재화 종류 및 정의

**[경험치]**
- 알 별 진화 경험치

**[공용 재화]**
- 알 육성 공용 재화

**[전용 재화]**
- 노란 알 재화
- 파란 알 재화
- 검정 알 재화
- 빨강 알 재화
- 하양/초록 알 재화

### 1.3. 재화 순환 구조

**러닝 게임 (재화 획득) → 육성 게임 (재화 소비)**

- 러닝 게임 플레이로 재화를 획득하여 육성 게임 내 육성 행동으로 재화를 소비한다.
- 각각의 알들은 러닝 게임을 통해 경험치와 육성 재화를 획득하여 진화를 할 수 있다.
- 진화를 위해서는 일정 시간동안 러닝게임 플레이 및 육성 재화 소비 행동을 n회 진행해야 한다.

## 2. 보상 계산 시스템

### 2.1. 보상 획득 기본 원리

- **거리 기반 보상**: 달린 거리에 비례하여 재화 획득
- **색상 별 차등 보상**: 고유 색상 vs 기타 색상으로 구분
- **시간 기반 경험치**: 플레이 지속 시간이 영혼의 성장을 의미
- **추가 보상**: 클리어 타임 별 보너스 배율, 클리어 보너스

### 2.2. 재화 획득 공식

**거리 계산 기준**
- 달린 거리 계산: 게임 내 1픽셀 = 0.1m (실제 거리 환산)
- 최소 측정 단위: 0.1m
- 측정 정밀도: 소수점 첫째 자리까지

| 재화 타입 | 타입 별 획득 계산 공식 |
|-----------|------------------------|
| **경험치** | • (플레이 시간(초) × 0.5) + 진행도 별 보너스<br>• 진행도 별 보너스: [20% 미만: +0], [20~39%: +5], [40~59%: +10], [60~79%: +20], [80~99%: +35], [100%(클리어): +50] |
| **공용 육성 재화** | • 기타 색상 달린 거리 × 기본 배율 × 보너스 배율<br>• 기본 배율: 10 (거리 1m당 10개)<br>• 보너스 배율: [클리어 보너스: 1.5배], [클리어 타임 보너스: 1.0~1.4배 (시간에 따라 차등)]<br>• 클리어 타임 보너스 배율: 목표시간 × n (목표 시간 별 보너스 배율 별도 테이블로 관리) |
| **전용 육성 재화** | • 고유 색상 달린 거리 × 기본 배율 × 희귀도 배율 × 보너스 배율<br>• 기본 배율: 15 (거리 1m당 15개)<br>• 희귀도 배율: [빛의 알 (노랑): 1.0배], [물의 알 (파랑): 1.1배], [어둠의 알 (검정): 1.2배], [불의 알 (빨강): 1.3배], [대지의 알 (초록): 1.4배]<br>• 보너스 배율: [클리어 보너스: 1.5배], [클리어 타임 보너스: 1.0~1.4배 (시간에 따라 차등)] |

## 3. 진행도 시스템

### 3.1. 진행도 개요

- 각 알 별로 1개의 맵을 가진다. (맵 구조-레벨 디자인 상이)
- 알 별 맵 클리어 %를 진행도라고 한다. (맵 클리어 = 진행도 100%)
- 진행도 100% 도달 시 다음 알의 맵이 오픈된다.

| 알 종류 | 맵 총 길이 | 예시 난이도 |
|---------|-----------|-------------|
| 빛의 알 (노랑) | 500m | ★☆☆☆☆ |
| 물의 알 (파랑) | 600m | ★★☆☆☆ |
| 어둠의 알 (검정) | 700m | ★★★☆☆ |
| 불의 알 (빨강) | 800m | ★★★★☆ |
| 대지의 알 (초록) | 900m | ★★★★★ |

### 3.2. 진행도 계산 방식

**진행도(%) = 현재 도달 지점 / 맵 총 길이**

- 맵 총 길이 = 사전 정의된 메인 루트의 총 거리
- 최단 경로가 아닌 "권장 경로"로 설정
- 각 알 별로 상이한 총 길이 (난이도 반영)

**진행도 표시 예시**
- 지나온 메인 루트: 흰색 선
- 가야 할 메인 루트: 노란 선
- 실제 게임에서는 메인 루트를 직접적으로 표현하지는 않는다.
- 시작점 > 도착점 까지의 메인 루트로 진행도 계산
- 예: 빛의 알 진행도 67% (시작점부터 진행도 55%, 탈출구까지 남은 진행도 45%, 100% 달성 시 오픈)

## 4. 재화 소비 시스템

### 4.1. 재화 소모 시스템 개요

- 알 육성 시 육성 행동을 선택할 수 있으며 선택한 행동 별 필요한 재화 및 필요량에 차이가 있다.
- 공용 재화의 경우 모든 알 육성 시 사용 가능하며 전용 재화의 경우 전용 알 육성에만 사용 가능하다.

### 4.2. 육성 행동 별 재화 소모량

| 육성 행동 | 공용 재화 | 특별 재화 | 증가하는 진화 스텟 |
|-----------|-----------|-----------|-------------------|
| 1. 밥 주기 | 50 | 0 | 온순함 |
| 2. 산책하기 | 20 | 5 | 활발함 |
| 3. 책 읽기 | 30 | 15 | 신중함 |
| 4. 두들겨보기 | 100 | 25 | 대담함 |
| 5. 노래 들려주기 | 40 | 20 | 예술적 |

> 육성 행동 명칭, 진화 스텟 수량과 명칭, 필요 재화량은 변경될 수 있다. (테이블 관리 필요)

### 4.3. 진화 조건

**진화 조건 = 경험치 100% + 육성 행동 n회 이상 진행**

- 경험치: 러닝 게임 플레이 유도 (알 별 차등)
- 육성 행동 n회: 육성 게임 플레이 유도 (알 별 차등)

## 5. 데이터 테이블

### 5.1. 재화 기본 테이블 – currency_table

| 칼럼 | 데이터 타입 | 설명 |
|------|-------------|------|
| Id | Int | 고유 번호 |
| Type | string | 재화 사용 pc 타입 / 0 = 공용 |
| Rarity | float | 희귀도, 재화 획득 시 해당 칼럼 값을 곱하여 획득한다.(소수점 이하 버림) |
| icon | string | 아이콘 리소스 연동 |

**테이블 데이터**

| Id | Type | Rarity | icon | 설명 |
|----|------|--------|------|------|
| 1 | 0 | 0 | Icon_currency_exp | 경험치 |
| 2 | 0 | 0 | Icon_currency_dew | 공용 재화 |
| 3 | yellow | 1 | Icon_currency_yellow | 노란 알 전용 재화 |
| 4 | blue | 1.1 | Icon_currency_blue | 파란 알 전용 재화 |
| 5 | black | 1.2 | Icon_currency_black | 검은 알 전용 재화 |
| 6 | red | 1.3 | Icon_currency_red | 빨간 알 전용 재화 |
| 7 | white | 1.4 | Icon_currency_white | 하얀 알 전용 재화 |

> 게임 내 인벤토리가 존재하지 않음으로 필요 정보는 보상 연동을 위한 id 값과 전용 알 연결 정보, 아이콘. 추가적인 정보가 필요하지 않고, 추후 추가적인 캐릭터 업데이트, 재화 업데이트를 데이터 제어로 하지 않아도 된다면 별도 테이블로 관리할 필요는 없을 것 같다.
